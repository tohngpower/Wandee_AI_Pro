# Wandee_AI_Pro
* Install Ollama
* Install Python
* Install VS code
* Open **CMD**

    > ollama run gemma3

    > /bye

* Close **CMD**
* Create folder **Gemma**
  
    > python -m venv ./env
    
* Open folder **Gemma** with Code
* Open terminal
  
    > ./env/Scripts/Activate
    
    > pip install ollama
    
    > pip install streamlit
    
* Close VS code
* Run **app.bat**
* Enjoy your local AI chatbot
